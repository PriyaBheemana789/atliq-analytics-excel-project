# atliq-analytics-excel-project
Sales, Financial and Market Analytics(Multi Dimensional Analysis) Project using Excel, Power Pivot, Pivot Tables, DAX, and Data Modeling for business performance analysis.

**Project Overview**
This project focuses on analyzing sales, profitability, and market performance for ATLIQ Hardware using Microsoft Excel, Power Pivot, and Pivot Tables. The objective was to build an interactive analytical reporting solution that helps business stakeholders track revenue trends, profitability, and target achievement across customers and markets.

**Business Objectives**
-Analyze customer-wise net sales performance
-Compare market performance against targets
-Evaluate yearly and monthly P&L trends
-Identify high-profit and low-profit markets
-Support data-driven business decision-making

**Tools & Technologies**
Microsoft Excel
Power Pivot
Pivot Tables
Power Query
DAX
Data Modeling

**Key Features**
Customer sales performance analysis
Market target comparison
Fiscal year P&L reporting
Monthly profitability tracking
Gross Margin and GM% calculations
Interactive filtering using Pivot functionality

**Technical Highlights**
Built relational data model using Power Pivot
Applied fiscal year logic and calculations
Designed multi-dimensional reporting structure
Created dynamic Pivot-based business reports
Performed data cleaning and transformation

**Key Insights**
Sales growth increased significantly in FY2021.
Certain markets generated strong revenue but lower profitability.
Revenue concentration among major customers indicated business dependency risks.
Market-level target analysis revealed underperforming regions.

**Business Impact**
This project demonstrates how Excel-based analytics can support:
Revenue tracking
Profitability optimization
Market performance monitoring
Executive reporting
Strategic business decisions
